# Raw Dataset Inputs

Place the raw dataset files here before running the workflow.

Expected local inputs:

- `OhioT1DM/`
- `WESAD.zip`
- `PPG_FieldStudy.zip`

Large zip files are ignored by Git because WESAD and PPG-DaLiA are several gigabytes. The processed tables, metrics, figures, and notebooks are stored inside `three_real_dataset_work`.
